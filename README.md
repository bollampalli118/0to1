# 💰 Wealth Tracker – Personal Finance Dashboard

A modern, interactive personal finance dashboard designed to track assets, liabilities, investments, and financial health in a single interface.

## 🚀 Overview

Wealth Tracker is a browser-based application that helps users monitor:

- Net worth (real-time)
- Assets (Gold, Land, Mutual Funds, PF, Cash, LIC)
- Liabilities (Loans, EMIs, Interest)
- Income vs Expenses
- Financial runway & projections

The application provides a **single source of truth for personal financial planning**.

---

## 🎯 Key Features

### 📊 Net Worth Dashboard
- Real-time calculation of total assets and liabilities
- Monthly income vs expense tracking
- Financial health indicators

### 🥇 Asset Management
- Gold valuation based on daily rate
- Land appreciation calculation
- Mutual fund tracking
- EPF & PPF projections
- LIC policy tracking

### 📉 Liability Tracking
- Gold loan interest calculations
- Personal loan EMI breakdown
- Outstanding balance tracking
- Debt-free projections

### ⚙️ Data Hub (Core Engine)
- Centralized input system
- Auto-updating calculations across all modules
- Real-time recalculation of all metrics

### 🧮 Financial Calculator
- EMI calculations
- Investment projections
- Loan planning tools
